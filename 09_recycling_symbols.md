# Lesson 09: Recycling Symbols

Many plastic products have a number inside a triangle.

People often call this a recycling symbol.

But the number does not guarantee that the item will be recycled.

This lesson explains what the number actually means.

## Why Care About Recycling Symbols?

Because the symbol is easy to misunderstand.

The number mostly identifies the resin type.

It does not automatically mean:

```text
your local program accepts it
the item will be sorted correctly
the item will be turned into a new product
the package is economically recyclable
```

The symbol is a material clue, not a recycling promise.

## Step 1: Resin Identification Codes

The numbers are resin identification codes.

They help identify broad plastic categories.

The common codes are:

```text
1 PET
2 HDPE
3 PVC
4 LDPE
5 PP
6 PS
7 Other
```

The code tells you the main resin family.

It does not tell you the full recipe, additives, color, contamination, or local
recycling rules.

## Step 2: Code 1 PET

Code 1 is PET.

Common examples:

```text
drink bottles
some food containers
```

PET bottles are one of the more commonly recycled plastic packaging types in
many systems.

But details still matter:

```text
color
labels
caps
food residue
local program rules
```

So do not stop at the number.

Check the local recycling instructions.

## Step 3: Code 2 HDPE

Code 2 is HDPE.

Common examples:

```text
milk jugs
detergent bottles
shampoo bottles
some rigid containers
```

HDPE bottles are also commonly accepted in many recycling programs.

Natural or lightly colored HDPE can be especially useful because it can be made
into more colors later.

But again:

```text
accepted locally?
clean enough?
sorted correctly?
```

Those questions still matter.

## Step 4: Code 3 PVC

Code 3 is PVC.

Common examples:

```text
pipes
flooring
cable insulation
some rigid packaging
```

PVC is not handled the same way as PET or HDPE bottles in many municipal
programs.

It has different chemistry, additives, and recycling challenges.

This is why resin type matters.

Mixing incompatible plastics can reduce recycled material quality.

## Step 5: Code 4 LDPE

Code 4 is LDPE.

Common examples:

```text
bags
films
wraps
squeeze bottles
```

LDPE film may require separate collection from rigid containers.

Many curbside systems do not want loose plastic bags or films because they can
tangle in sorting equipment.

Some stores or drop-off programs may accept certain clean films.

The key lesson:

```text
same material family does not mean same recycling pathway
```

## Step 6: Code 5 PP

Code 5 is PP.

Common examples:

```text
caps
tubs
food containers
hinged lids
```

PP recycling acceptance varies by location.

It has become more common in some systems, but local rules still decide.

PP is useful because it is lightweight, tough, and handles heat better than some
common packaging plastics.

## Step 7: Code 6 PS

Code 6 is PS.

Common examples:

```text
rigid disposable items
foam packaging
foam food containers
```

Polystyrene can be difficult in recycling systems, especially when foamed.

Foam is bulky and light, which can make collection and transport inefficient.

Food contamination can make it harder.

This is why some items may have a resin code but still not be accepted locally.

## Step 8: Code 7 Other

Code 7 means other.

It can include:

```text
polycarbonate
nylon
PLA
multi-layer packaging
mixed plastics
materials outside codes 1-6
```

This category is very broad.

Code 7 does not tell you one specific material.

It tells you the item does not fit cleanly into the first six categories.

That can make recycling more complicated.

## Step 9: The Practical Rule

Use the resin code as the first clue.

Then ask:

```text
does my local program accept this shape and material?
is the item clean?
is it a bottle, tub, film, foam, or mixed package?
are caps or labels handled separately?
```

Recycling is local because collection, sorting equipment, buyers, and economics
are local.

The best rule is:

```text
resin code tells material family
local program tells what to do
```

## Step 10: Three Quick Questions

Question:

```text
Does a resin code guarantee that an item will be recycled?
```

Answer:

```text
no
```

Question:

```text
What does code 1 usually mean?
```

Answer:

```text
PET
```

Question:

```text
Why should you check local recycling rules?
```

Answer:

```text
accepted materials and sorting systems vary by location
```

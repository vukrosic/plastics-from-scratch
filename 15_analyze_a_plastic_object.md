# Lesson 15: Analyze A Plastic Object

This capstone ties the course together.

The goal is to look at a plastic object and explain it like a materials
engineer.

Not perfectly.

Not with a laboratory.

Just with a clear model.

## Why Care About Object Analysis?

Because plastics become real through objects.

The course ideas are useful when you can apply them to:

```text
a bottle
a cap
a bag
a food tray
a phone case
a pipe
a wrapper
a container
```

The object in your hand contains decisions about material, process, properties,
cost, and end-of-life.

## Step 1: Identify The Object And Job

Start with the product's job.

Ask:

```text
what does this object need to do?
what does it hold or protect?
does it need to bend?
does it need to be clear?
does it need to resist heat?
does it need to seal?
does it need to be cheap?
does it need to be reused?
```

Example:

```text
clear drink bottle:
    hold liquid
    resist impact
    stay lightweight
    show contents
    seal with cap
    survive transport
```

The job explains many material choices.

## Step 2: Look For Material Clues

Look for:

```text
resin code
label
markings
transparency
flexibility
stiffness
surface feel
color
foam structure
layers
```

A resin code can help, but remember:

```text
resin code = clue
not full recipe
not recycling guarantee
```

If there is no code, use the object's properties and use case as clues.

You are building a hypothesis, not proving chemistry.

## Step 3: Guess The Polymer Family

Use the common material map:

```text
clear drink bottle:
    likely PET

milk jug:
    likely HDPE

thin grocery bag:
    likely LDPE or related film material

bottle cap:
    often PP or HDPE

foam packaging:
    often PS or another foamed polymer

rigid pipe:
    often PVC
```

Write the guess as:

```text
likely material:
    PET

confidence:
    medium

reason:
    clear bottle, common application, resin code if present
```

That keeps your thinking honest.

## Step 4: Infer The Manufacturing Process

Use shape clues.

Ask:

```text
is it hollow?
is it thin film?
is it a precise molded part?
is it a continuous shape?
is it a sheet formed into a tray?
is it foamed?
```

Process guesses:

```text
hollow bottle:
    likely blow molded

cap:
    likely injection molded

pipe:
    likely extruded

thin bag:
    likely film extrusion

food tray:
    likely thermoformed

foam packaging:
    foaming process
```

Manufacturing explains shape.

Shape gives clues about manufacturing.

## Step 5: Explain The Properties

Describe what you can observe:

```text
clear or opaque
flexible or stiff
tough or brittle
smooth or textured
thin or thick
light or heavy
heat-resistant or heat-sensitive
single material or multi-part
```

Then connect properties to structure:

```text
flexible film:
    chains can move enough for bending

rigid container:
    structure resists deformation

foam:
    gas pockets reduce density

clear bottle:
    structure lets light pass with low scattering
```

You are practicing:

```text
structure -> properties -> use
```

## Step 6: Think About Additives And Design Details

Ask:

```text
is it colored?
does it need UV resistance?
does it need flexibility?
does it need fire resistance?
does it have fillers?
does it have labels or adhesives?
does it have multiple parts?
```

You may not know the exact additives.

That is fine.

The point is to recognize that the final object may be a formulation, not a pure
polymer.

Example:

```text
black plastic housing:
    likely pigment
    maybe fillers
    maybe flame retardant if used in electronics
```

## Step 7: Think About End Of Life

Ask:

```text
is it likely accepted locally?
is it clean?
is it a film, foam, bottle, tub, or mixed package?
does it have multiple materials?
are labels or caps compatible?
is reuse possible?
would recycling be mechanical, chemical, composting, or disposal?
```

Do not say:

```text
it has a symbol, so it is recyclable
```

Say:

```text
the symbol suggests a resin family
local systems determine what happens next
```

That is much more accurate.

## Step 8: Example Analysis

Object:

```text
clear disposable water bottle
```

Job:

```text
hold water
stay lightweight
resist breaking
show contents
seal with cap
survive transport
```

Likely material:

```text
PET bottle body
PP or HDPE cap
paper or plastic label
adhesive
```

Likely process:

```text
bottle:
    blow molding from a preform

cap:
    injection molding
```

Properties:

```text
clear
light
tough enough for transport
thin-walled
not meant for high heat
```

End-of-life:

```text
often recyclable where PET bottles are accepted
depends on local program
cleanliness, labels, caps, and sorting still matter
```

Tradeoff:

```text
very lightweight and useful
often short-lived
requires collection and recycling system to avoid waste
```

## Step 9: Analysis Template

Use this template for any object:

```markdown
Object:

Job:

Likely polymer:

Evidence:

Likely manufacturing process:

Important properties:

Additives or design details:

End-of-life pathway:

Main tradeoff:
```

The goal is not to be perfect.

The goal is to explain your reasoning.

## Step 10: Three Quick Questions

Question:

```text
What should you identify before guessing the material?
```

Answer:

```text
the object's job
```

Question:

```text
Why is a resin code not the full answer?
```

Answer:

```text
it is a material clue, not the full recipe or recycling guarantee
```

Question:

```text
What is the main thinking pattern for analyzing a plastic object?
```

Answer:

```text
structure -> properties -> use -> end of life
```
